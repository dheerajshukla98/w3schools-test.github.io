Git & GitHub Commands

Git Configuration

git config --global user.name "Your Name" - Set user name

git config --global user.email "your-email@example.com" - Set user email

Initializing a Repository

git init - Initialize a new Git repository

git clone <repository-url> - Clone an existing repository

Basic Git Commands

git status - Check repository status

git add . - Stage all changes

git commit -m "Your commit message" - Commit changes

git push origin main - Push changes to remote repository

git pull origin main - Pull latest changes from remote

Branching & Merging

git branch - List branches

git branch new-branch - Create a new branch

git checkout new-branch - Switch to a branch

git merge new-branch - Merge a branch into the current branch

Undoing Changes

git reset --hard HEAD - Reset to last committed state

git checkout -- filename - Discard changes in a file

GitHub Collaboration

git remote add origin <repository-url> - Add remote repository

git push -u origin main - Push code to GitHub

git fetch - Fetch latest changes

git pull - Pull updates from remote
